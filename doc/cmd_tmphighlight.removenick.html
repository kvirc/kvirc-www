tmphighlight.removeNick

Remove a user from the channel temporary highlight list
Usage 
tmphighlight.removeNick <nick>
Description
This command remove a user from the channel temporary highlight list, so that user messages
stop being highlighted.
Syntax Specification 
tmphighlight.removeNick <nick:string>
See also
