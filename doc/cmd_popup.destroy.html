popup.destroy

Destroies a popup
Usage 
popup.destroy [-q] <popupname>
Description
Destroys a popup named <popupname>. To destroy a nested popup, use popup.delitem instead.
Switches
-q | --quiet
Run quietly: don't print warning and errors
Syntax Specification 
popup.destroy [-q] <popupname:string>

defpopup, popup.show, popup.create, $popup.exists
