Array concatenation operator
This operator concatenates arrays The syntax is:

<target> <+ <right_operand>

If <target> is not an array, it is converted to one first. After that, if <right_operand> is a scalar then it is appended to the end of the <target> array. If <right_operand> is an array then all of its items are appended to the end of the <target> array. If <right_operand> is a hash then all of its value items are appended to the end of the <target> array.
