The reguser module

Interface to the registered users database
The reguser module is the scripting interface to the registered users database.
It provides the following set of commands:
reguser.add: adds an user entry to the database
reguser.addmask: adds an user's mask to the database
reguser.remove: removes an user entry from the database
$reguser.list: lists the entries
reguser.setproperty: sets a property associated to an entry
reguser.edit: shows an editor for an user entry $reguser.match(): finds an entry by matching masks
$reguser.exactmatch(): finds an entry by comparing masks
$reguser.property(): finds a property associated to an entry
$reguser.matchproperty(): shortcut for $reguser.property($reguser.match())
$reguser.mask(): returns the registration masks for an entry
