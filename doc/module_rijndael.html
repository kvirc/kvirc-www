The rijndael module

The Rijndael cryptographic engines
The rijndael module exports six cryptographic engines based on the Advanced Encryptiong Standard algorithm called Rijndael. Rijndael was originally written by Joan Daemen and Vincent Rijmen. The original Rijndael description is available at http://www.esat.kuleuven.ac.be/~rijmen/rijndael/.
It is a private key block cipher that has been designed to replace the widely used DES, and it should provide at leas a decent security agains common attacks. Theoretically the best attack that one can perform on this cipher is the "brute force" attack that requires a really massive parallel computation: actually out of the possibilities of a common "hacker".
My implementation allows the usage of 128, 192 and 256 bit keys on 128 bit data blocks. The encrypted binary data buffer is then converted into an ascii-string by using the base64 conversion or hex-digit-string representation. The six engines are the six possible combinations of the key lengths and ascii-string conversions.
