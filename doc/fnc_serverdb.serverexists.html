$serverdb.serverExists

Checks if the network already exists in the DB
Usage 
$serverdb.serverExists(<servername>[,<networkname>])
Description
Checks if the server already exists for a network in the DB.
If no network name is provided, the check is made globally.
It returns 1 if the server exixts, 0 otherwise
Syntax Specification 
<bool> $serverdb.serverExists(<servername:string>[,<networkname:string>])
