$mediaplayer.position

Returns the position of the current media
Usage 
$mediaplayer.position()
Description
Returns the currently elapsed time of the multimedia file being played by the remote instance of the media player. The time is expressed in milliseconds.
Take a look at the mediaplayer module documentation for more details about how it works.
Syntax Specification 
$mediaplayer.position()
