OnDCCSessionDestroyed

A new DCC session has just been destroyed
Parameters 
$0 = DCC session ID
Window
console
Description
Triggered when a new DCC session has just been destroyed. You can retrieve the session information by calling the dcc.* module functions and by passing the DCC session ID parameter.
