$typeof

Returns the internal data type of a variable
Usage 
$typeof(<data>)
Description
Returns the internal data type of the <data>.
Syntax Specification 
<string> $typeof(<data:variant>)
Examples 

    echo $typeof("test")
    %a = "test"
    echo $typeof(%a)
    %a = 1
    echo $typeof(%a)
    %a = $(1 + 2)
    echo $typeof(%a)
    echo $typeof($typeof(%a))
