OnIRCContextDestroyed

Triggered when an IRC context is destroyed
Parameters 
None
Window
Console of the destroyed IRC context
Description
This event is triggered when an IRC context is being destroyed. It is bound to the console of the related IRC context, that is the last window of the context that is destroyed.
