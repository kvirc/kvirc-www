serverdb.addServer

Adds a server
Usage 
serverdb.addServer [switches] <string> <string>
Description
Adds the server <server> to the network <network>.
Switches
-a | --autoconnect
Autoconnect this server at KVIrc start.
-c | --cache-ip
Caches the server IP at first connect to save bandwitch for future connects.
-i | --ipv6
Use IPv6 socket to connect to the server.
-p=<port> | --port=<port>
Use the port <port> to connect to the server.
-q | --quiet
Do not print errors if the server already exist.
-s | --ssl
Use SSL (Secure Socket Layer) socket to connect to the server.
-w=<password> | --password=<password>
Use password <password> to connect to the server.
Syntax Specification 
serverdb.addServer [switches] <string:network> <string:server>
Examples 

Sets the nickname HelLViS69 for the server irc.freenode.net

serverdb.setNickName -s irc.freenode.net HelLViS69
