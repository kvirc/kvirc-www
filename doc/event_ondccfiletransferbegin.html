OnDCCFileTransferBegin

Triggered when a DCC File transfer begins
Parameters 
$0 = DCC session ID
Window
unspecified
Description
Triggered when a DCC file transfer connection is established and the real data transfer starts.
You can retrieve the session information by calling the dcc.* module functions and by passing the DCC session ID parameter.

ondccfiletransferconnectioninprogress, ondccfiletransfersuccess, ondccfiletransferfailed
