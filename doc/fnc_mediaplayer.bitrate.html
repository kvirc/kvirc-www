$mediaplayer.bitRate

Returns the bit rate of the current song
Usage 
$mediaplayer.bitRate
Description
Returns the bit rate (in bits/sec) of the multimedia file being played by the remote instance of the media player.This function is not granted to work with all the player interfaces and all the media types so you may get -1 or 0 in return sometimes.
Take a look at the mediaplayer module documentation for more details about how it works.
Syntax Specification 
$mediaplayer.bitRate
