squery

Sends a message to a service
Usage 
squery <target> <text>
Description
Sends a message to a service in a form similar to privmsg.
<target> must be a service(see RFC2812). This command is connection dependant.
Switches
-q | --quiet
Do not print any output
Syntax Specification 
squery <target> <text>
Examples 

squery Alis LIST * -min 100
